# LuaAnsi
## Syntax
Here's an example:
```
This is $b[bold and $i[italic]]! This is also $bi[bold and italic]
Here's an example for [brackets$]
```
This is **bold and *italic***! This is also ***bold and italic***  
Here's an example for \[brackets\]
| Character | Format |
| --- | --- |
| `b` | bold |
| `i` | italic |
| `u` | underline |
| `s` | strikethrough |
